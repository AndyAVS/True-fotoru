## Кастомизация форума fotoru

### Зачем?
Скрываем посты некоторых пользователей. Убираем артефакты "игнор-листа": когда заигнорили нас. Декоративно, но возвращаем некоторые кнопки интерфейса форума. Глубоко не рыл. Устанавливаем стили форума, пригодные для просмотра фото.

### Как установить
* Стянуть к себе  **True-Fotoru.user.js**  из последнего [release](https://github.com/AndyAVS/true-fotoru/releases/) Tampermonkey, если установлен, подхватит его.
